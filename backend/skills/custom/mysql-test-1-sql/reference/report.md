# Review & Report Rules

After getting query results, provide analysis in this format:

## Analysis Report
1. **Summary**: One-sentence answer to the user's question
2. **Key Findings**: Bullet points of important data points
3. **Data Details**: Present relevant numbers with context
4. **Methodology**: Brief description of the query approach
5. **Caveats**: Any data quality issues, assumptions, or limitations

## Formatting
- Use tables for structured data comparisons
- Round decimals appropriately (currency: 2 decimals, percentages: 1 decimal)
- Include units (currency symbol, %, etc.)
- For time series data, note the time range and granularity
